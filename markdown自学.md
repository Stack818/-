### 流程图

```mermaid
graph LR
A-->B
B-->C
C-->A

```

> LR 表示流程图从左向右；下面做解释

| 用词 | 含义     |
| ---- | -------- |
| TB   | 从上到下 |
| BT   | 从下到上 |
| LR   | 从左到右 |
| RL   | 从右到左 |

#### 常用符号

| 表述       | 说明           | 含义               |
| ---------- | -------------- | ------------------ |
| id[words]  | 矩形节点       | 过程，流程中的环节 |
| id(文字)   | 圆角矩形节点   | 开始或结束         |
| id((文字)) | 圆形节点       | 连接               |
| id{文字}   | 菱形节点说明   | 判断，决策含义     |
| id>文字]   | 右向旗帜型节点 |                    |
